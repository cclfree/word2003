크리에이티브 커먼즈 라이선스
이 저작물은 크리에이티브 커먼즈 코리아 저작자표시-비영리-동일조건변경허락 2.0 대한민국 라이선스에 따라 이용하실 수 있습니다.

1.한글입력기 바람 http://baramim.blogspot.com/

바람은 열린한글프로젝트의 한글라이브러리(libhangul 0.8)에 기반한 Mac OS X용 한글입력기 입니다. 바람은 한영 전환 단축키와 한자입력기능을 가지고 있으며 다양한 한글 자판을 지원합니다. 또한, “바람 사전”을 통해서 사용자 사전을 만들거나 편집할 수 있습니다.


주요 기능

  1. 다양한 한영 전환 방법 지원 (Shift-space 및 오른쪽 특수키 지원)
    2. 모아쓰기를 지원하여 오타를 최소화 (libhangul 기능)
  2. 자동 입력 모드 전환 기능
    4. 프로그램 별 시작 언어 설정 기능

    5. 한자어 입력 가능
  3. 한자 사전을 통한 사용자 상용구 지원
  4. MS 윈도우 호환 특수문자 입력 기능



    시스템 요구사항

    바람은 Mac OS X 10.5 이상에서만 정상동작합니다.

    PPC와 Intel 프로세서에서 모두 동작하는 Universal 코드입니다.


    2.Movist http://cocoable.tistory.com/

    무비스트는 사용 편의성과 고품질 자막, 자체 코덱 내장 등을 주요 목표로 개발 중인 Mac OS X용 동영상 플레이어 입니다. 아직 개발 초기라서 부족한 점이 많지만 훌륭한 동영상 플레이어로 발전할 수 있도록 많이 사랑해주시기 바랍니다. 무비스트는 프리웨어(freeware) 입니다.


    주요 기능 및 특징

    • QuickTime 및 FFmpeg 지원

    • 다양한 설정의 고품질 자막 표시

    • smi, srt 자막 형식 지원

    • 재생 목록 지원

    • 동영상 제어창 지원

    • 전체 화면 제어창 지원

    • 전체 화면 탐색 지원

    • 다양한 사용 편의성

    • 유니버설 바이너리


    실행 환경

    • Mac OS X 10.4 Tiger, 10.5 Leopard

    • Intel 및 PowerPC G4 이상 매킨토시
    3.네이트온 http://nateonweb.nate.com/help/maclinux/mac_download.htm


    4.KMLE 통합 사전 1.1 http://www.kmle.co.kr/board/bbs/board.php?bo_table=kmle&wr_id=5491

    ◈ 프로그램 기능


    영한/한영 사전 124,188 단어, 영영 사전 195,817, 의학 약어 20,956 단어 통합 검색

    별도로 검색 버튼이나 엔터를 누르지 않아도 타자치면서 실시간 검색

    “검색어”로 시작하는 모든 단어 검색 기능 별도 제공





    ◈ 지원 운영체제(OS)


    윈도우(Win32 호환: Windows 95, 98, 2000, 2003, XP, Vista) - 자동 인스톨러 지원

    Mac OS X(Apple), Linux, Solaris(SPARC) - 자바로 실행 가능, 기타 JAR 데이터베이스 파일 다운로드



    5.맥 os x 용 티스토리 알리미 티돌이 http://www.cocoadev.co.kr/53

    OS X에서 티스토리 계정 블로그의 새로운 댓글, 피드백, 방명록이 작성되었을 때 알려주는 프로그램입니다.

    사용환경은 유니버셜 바이너리 (PPC/Intel 사용 사용 가능), OS X 10.4 (타이거) 이상 입니다.


    6.국내 음악 가사를 찾아주는 ‘가사수작’ http://kiding.kmug.co.kr

    이 프로그램은 ‘가사수작’이라는 애플스크립트 기반 프로그램으로, iTunes에서 현재 재생되고 있는 곡을 검색하여 가사 및 앨범 사진을 찾아주는 프로그램입니다.


    사용법은 간단합니다. iTunes에서 음악을 재생시키고, 프로그램을 실행하면 가사를 찾기 시작합니다. 처음은 제목+아티스트명으로 찾고, 없으면 제목만으로 검색하여 가장 근접한 결과를 보여줍니다.


    7.자막도 넣어주는 아이팟 인코더 ‘팟벗’ http://kiding.kmug.co.kr

    자막도 인코딩 해주는 mencoder+ffmpeg+mp4box 프론트엔드 ‘팟벗 (PodBuddy)’

    맥에선 처음으로 한글 SMI(+SRT) 자막을 씌워 아이팟용 영상으로 만들어주는 프로그램


    8.한국 날씨 표시 widget 바깥 날씨 http://appletree.or.kr/widgets/WeatherOutside/

    바깥 날씨는 한국 기상청의 현재 날씨 정보와 매분관측자료에서 날씨 정보를, 그리고 주요도시예보에서 5 일간의 예보 정보를 가져와서 Dashboard의 화면 위에 날씨 정보를 보여주는 한국 날씨 표시 widget입니다.



    한국 기상청에서 날씨 정보를 직접 가져오므로 Mac OS X 10.4 Tiger에서 기본으로 제공되는 날씨 widget보다 비교적 더 정확한 현재 날씨 정보를 보여주며, 아울러 더 많은 지역들의 날씨 정보를 제공합니다.


    9.워드프로세서 MyPage http://www.da-soft.co.kr/bbs/zboard.php?id=MyPage&no=1

  • 자간, 행간, 베이스라인, 좌우 여백, 단락 앞뒤 여백 조정기능
  • 그림자체, 밑줄체, 텍스트중심선체, 볼드체, 이탤릭, 장평 조정, 외곽선체 지원, 투명 칼라 지원

  • 투명 기능

  • 다양한 선 형태와 바탕색 처리 기능, 그림자 효과
  • 이미지 확대/축소, 회전, 이동
  • 다양한 도형 효과(원형, 직선, 화살표, 사각형, 별, 다각형등)
