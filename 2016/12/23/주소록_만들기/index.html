주소록 만들기

2016-12-23
  • 만들기
  • 주소록

실습자료 :

address.xls

먼저 ‘학번’ 순으로 정렬하고, 같은 학번일 경우 ‘이름’ 순으로 정렬하고 싶은데, 어떻게 해야 하나요?

주소록 ‘학번’ 필드명 셀에 셀 포인터 위치시키고, [데이터 - 정렬]. 정렬 기준으로 ‘학번’, ‘기준 추가’ 후 다음 기준으로 ‘이름’을 선택합니다. 2003의 경우, 첫째 기준으로 ‘학번’, 둘째 기준으로 ‘이름’을 선택합니다.

‘이름’ 필드 앞에 ‘번호’ 필드를 넣고 싶은데, 어떻게 해야 하나요?

‘이름’ 필드가 있는 A열 전체를 선택하고, 우클릭 후, ‘삽입’을 합니다.

필드명이 있는 첫 번째 행을 빼고, 나머지 2번 행에서 51번 행까지 행 높이를 좀 여유 있게 높게 하고 싶은데, 어떻게 해야 하나요?

2번 행번호부터 51번 행번호까지 행 전체를 선택하고, 우클릭 후, ‘행 높이’를 실행합니다. 행 높이로 ‘20’을 줍니다.

필드명 위에 주소록 제목으로 ‘동창회 주소록’이라고 멋지게 적고 싶은데, 어떻게 해야 하나요?

우선 넣을 공간을 만들고, ‘삽입’ 메뉴의 ‘WordArt(워드아트)’를 실행합니다(2003의 경우 [삽입 - 그림 - WordArt]입니다). 워드아트를 이동하려면 개체 테두리 선을 드래그 합니다. 워드아트의 모양을 바꾸려면, 워드아트 위에서 우클릭 후 ‘텍스트 효과 서식’을 실행합니다(2003의 경우 그림자를 넣으려면, 그리기 도구에서 ‘그림자 스타일’을 실행해야 합니다).

인쇄될 모양을 보려면 어떻게 하나요?

파일 - 인쇄.

두 번째 페이지에도 필드명이 표시되도록 하려면 어떻게 해야 하나요?

페이지 레이아웃 - 인쇄 제목. ‘시트’ 탭에서 ‘반복할 행’ 입력칸 클릭 후, 필드명 있는 셀 클릭합니다.

같은 학번끼리는 셀을 하나로 병합하고 세로쓰기를 하고 싶은데, 어떻게 해야 하나요?

같은 학번들을 선택하고 ‘셀 서식’을 실행합니다. ‘맞춤’ 탭에서 ‘셀 병합’을 체크하고 ‘방향’을 세로 텍스트로 선택합니다.

인쇄할 때 주소록을 페이지 좌우 가운데에 놓고 싶은데, 어떻게 해야 하나요?

<?xml:namespace prefix = “v” /><?xml:namespace prefix = “o” /><?xml:namespace prefix = “w” />[페이지 레이아웃 - 페이지 설정]의 대화 상자 시작 관리자()를 실행합니다(2003은 [파일 - 페이지 설정]). ‘여백’ 탭에서 ‘페이지 가운데 맞춤’의 ‘가로’를 체크합니다.

적정한 예산액을 어떻게 구하나요?
수입총액 예산 비율. =$B$1B5.

예산과 지출의 대차대조는 어떻게 계산하나요?
적정 예산액 - 수입액. =C5-E5.

대차대조 결과 금액이 음수(-)이면, ‘예산초과’, 양수(+)이면, ‘OK’를 자동으로 입력하려면 어떻게 해야 하나요?
IF 함수를 이용하여, 대차대조가 0보다 크면 ‘OK’, 그렇지 않으면 ‘예산 초과’. =IF(F5>0,”OK”,”예산초과”)

‘예산초과’ 글자색을 빨간 색으로 자동으로 바꾸려면 어떻게 하나요?
‘조건부 서식’을 사용합니다. ‘셀 값이’(‘셀 강조 규칙’) ‘다음값과 같음’(‘같음’) ‘예산초과’. ‘서식’에서 글자색으로 빨간 색(‘빨간 텍스트’).
